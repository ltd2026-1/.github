## Descripción General

**BizAnalyticsAI** es una plataforma SaaS de nivel empresarial construida para emprendedores, analistas de negocios y empresas en fase de crecimiento que necesitan inteligencia operacional y predictiva profunda. La plataforma combina un potente dashboard web con un ecosistema de ocho productos complementarios — aplicaciones de escritorio, apps móviles, extensiones de IDE y librerías para desarrolladores — todos integrados mediante un backend Supabase compartido con seguridad Row Level Security completa.

La plataforma está disponible en [bizanalyticsai.com.br](https://bizanalyticsai.com.br) y sirve tres portales distintos: el dashboard principal del usuario (`/dashboard`), un panel de control administrativo (`/admin`) y un portal de colaboración para estudiantes (`/alunos`).

### Diferenciadores Clave

| Funcionalidad | Descripción |
|---|---|
| **Analítica con IA** | Simulaciones Monte Carlo, modelado de Churn/LTV y pronóstico predictivo de ingresos |
| **Constructor de Plan de Negocios** | Creación estructurada de planes de negocio asistida por IA con exportación a PDF |
| **Motor de Viabilidad Financiera** | Análisis de punto de equilibrio, período de retorno y calculadoras de margen operacional |
| **Inteligencia de Mercado** | TAM/SAM/SOM, segmentación de industria, 5 Fuerzas de Porter y mapeo competitivo |
| **Gestión de KPIs** | Seguimiento de indicadores clave en tiempo real con definición de metas y benchmarks |
| **Ecosistema Multi-Producto** | 8 productos complementarios: escritorio, móvil, extensiones de IDE, extensión de navegador, librería Python |
| **White-Label por Usuario** | Cada usuario personaliza su dashboard — colores, marca, orden de navegación — con aislamiento total |
| **Arquitectura Tri-Portal** | Portales separados e independientemente seguros para usuarios, admins y estudiantes |
| **Multilingüe** | Soporte nativo para Portugués (pt-BR), Inglés (en-US) y Español (es-ES) |
| **PWA Completo** | Capacidad offline completa con service worker, manifiesto y prompt de instalación |
| **OpenAPI** | Spec OpenAPI generada automáticamente en `/api/v1/openapi` |

#### Tutorial de Onboarding

Los visitantes por primera vez ven un popup guiado de 20 pasos:

- Un paso por ruta del dashboard con ícono dedicado, color en gradiente y descripción
- Barra de progreso animada y navegador de puntos
- Lista de funcionalidades y callout "Consejo Pro" por paso
- Botones **Anterior / Siguiente** — cada Siguiente navega a esa ruta
- Finalización persistida en `localStorage` — nunca se muestra de nuevo tras la primera ejecución

### Panel Administrativo

Protegido en `/admin/*`. La sesión debe satisfacer `is_admin() = true` en Supabase.

> **Importante:** El layout admin **no** llama `signOut()` en acceso no autorizado — solo redirige a `/admin/login`. Esto previene destruir la sesión `/dashboard` del usuario al navegar entre portales.

## Despliegue

### Vercel (Recomendado)

Conecta el repositorio GitHub a Vercel para despliegues automáticos en producción en cada push a `main`.

```bash
npm i -g vercel
vercel --prod
```

**Configuración Vercel:**
- Preset de framework: **Next.js** (detectado automáticamente)
- Versión de Node.js: **20.x**
- Agrega todas las variables de entorno en **Configuración del Proyecto → Variables de Entorno**

### Supabase

1. Crea un proyecto en [supabase.com](https://supabase.com)
2. Abre el **SQL Editor** y ejecuta cada migración en `supabase/migrations/` en orden
3. Habilita **Row Level Security** en todas las tablas
4. Agrega tu dominio Vercel a las URLs de Redirect en **Authentication → Configuración de URL**
5. Copia tu **URL del Proyecto** y **anon key** a `.env.local`
